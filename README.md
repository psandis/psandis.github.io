# psandis.github.io

![psandis.github.io](public/projects/psandis.github.io.png)

Personal portfolio site for Petri Sandholm, engineering leader with 20+ years of experience in telecom, banking, and large-scale platforms.

## Tech Stack

- React + TypeScript
- Vite
- Three.js / React Three Fiber (3D carousel)
- Framer Motion (animations)
- Tailwind CSS

## Features

- Interactive 3D project carousel with auto-rotation
- Projects fetched live from GitHub API
- Project screenshot thumbnails on cards with back-of-card mirror reflection
- Language icon fallback for projects without screenshots
- Project detail modal with thumbnail, tech stack, and links
- Responsive header with hamburger menu on mobile
- Animated entrance, background marquee, and floating header
- Nordic Frost color theme

## Development

```bash
npm install
npm run dev
```
